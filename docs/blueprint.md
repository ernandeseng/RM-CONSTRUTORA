# **App Name**: R&M Construtora

## Core Features:

- Informative Home Section: Showcase R&M's vision, mission, and core values to build trust and credibility upon landing on the home section.
- Service Offerings Display: A clear and concise description of services like construction, renovations, and structural projects.
- Interactive Project Portfolio: Interactive showcase of completed construction projects with filtering and lightbox options. Add a generative AI 'tool' that recommends similar projects based on user preferences.
- Client Testimonials: Carousel of client testimonials to highlight satisfaction and project success.
- Contact and Inquiry Forms: Contact form for potential clients to submit inquiries, along with direct contact information.
- Direct Communication Channels: WhatsApp buttons linking directly to R&M's contact numbers for instant communication.

## Style Guidelines:

- Primary color: Petroleum Blue (#1B4B5A) to convey professionalism and trust.
- Secondary color: Lime Green (#C8D96F) for calls to action and modern highlights.
- Accent color: Metallic Gray (#808080) for construction-like detailing, providing depth and contrast. Lighter tones (#F5F5F5) provide a background.
- Headline font: 'Montserrat' (sans-serif) for titles to ensure modern robustness. Note: currently only Google Fonts are supported.
- Body font: 'Roboto' (sans-serif) for clear and accessible body text. Note: currently only Google Fonts are supported.
- Line-art or flat design icons related to construction and building for visual communication.
- Use angular section dividers and geometric shapes (gears, stylized buildings) to enhance the theme.
- Subtle fade-in and slide-up animations on scroll, enhancing user engagement.
